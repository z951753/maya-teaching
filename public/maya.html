命令参考

常用Maya命令的详细说明和使用方法

polyCube

建模
MEL

创建多边形立方体

语法

polyCube [-width #] [-height #] [-depth #] [-subdivisionsWidth #] [-subdivisionsHeight #] [-subdivisionsDepth #] [-axisX #] [-axisY #] [-axisZ #] [-createUVs string] [-name string]

示例

polyCube -w 10 -h 20 -d 5 -sw 2 -sh 4 -sd 2

sphere

建模
MEL

创建球体

语法

sphere [-radius #] [-subdivisionsX #] [-subdivisionsY #] [-axisX #] [-axisY #] [-axisZ #] [-createUVs string] [-name string]

示例

sphere -r 8 -sx 20 -sy 15

setKeyframe

动画
MEL

设置关键帧

语法

setKeyframe [-breakdown boolean] [-controlPoints boolean] [-hierarchy string] [-insert boolean] [-time string] [-value #] [-attribute string] [-channelBox boolean] [-shape boolean] objects

示例

setKeyframe -attribute "translateX" -time 1 -value 0 pCube1

cmds.polySphere

建模
Python

Python版本的创建球体命令

语法

cmds.polySphere(radius=1.0, subdivisionsX=20, subdivisionsY=20, axis=(0, 1, 0), name='sphere1')

示例

import maya.cmds as cmds cmds.polySphere(r=5, sx=16, sy=12, name='mySphere')

脚本示例

实用的MEL和Python脚本示例

建模脚本

批量创建几何体

Python
import maya.cmds as cmds

# 创建多个立方体
for i in range(5):
    for j in range(5):
        cmds.polyCube(name=f'cube_{i}_{j}')
        cmds.move(i*2, 0, j*2, f'cube_{i}_{j}')

print('创建完成！')

自动平滑网格

MEL
// 自动平滑选中的网格
string $sel[] = `ls -sl`;
for ($obj in $sel) {
    polySoftEdge -a 180 $obj;
    print ($obj + " 已平滑\n");
}
print ("平滑完成！");

动画脚本

批量设置关键帧

Python
import maya.cmds as cmds

# 为选中的对象设置关键帧
selected = cmds.ls(selection=True)
if selected:
    for obj in selected:
        # 设置 translateX 关键帧
        cmds.setKeyframe(obj, attribute='translateX', time=1, value=0)
        cmds.setKeyframe(obj, attribute='translateX', time=10, value=10)
        # 设置 translateY 关键帧
        cmds.setKeyframe(obj, attribute='translateY', time=1, value=0)
        cmds.setKeyframe(obj, attribute='translateY', time=10, value=5)
    print('关键帧设置完成！')
else:
    print('请先选择对象！')

工具脚本

清除无用节点

MEL
// 清除场景中的无用节点
string $unused[] = `ls -type "unknown"`;
for ($node in $unused) {
    delete $node;
}

// 清除空组
string $emptyGroups[] = `ls -empty -g`;
for ($group in $emptyGroups) {
    delete $group;
}

print("无用节点已清除！");
