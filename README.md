# Maya三维动画讲课平台

专为教师打造的备课资料管理系统，让您的Maya教学更高效、更专业。

## 项目简介

本系统是一个基于Web的Maya三维动画教学管理平台，旨在帮助教师更有效地整理和管理备课资料，提高教学质量和效率。系统支持课程管理、资料管理、教学笔记、Maya专用工具等功能，为教师提供一站式的教学资源管理解决方案。

## 核心功能

### 1. 课程管理
- 创建和管理Maya相关课程（建模、动画、材质、灯光等）
- 编辑课程信息和大纲
- 课程分类管理（基础理论、高级技巧、案例分析等）
- 课程进度追踪

### 2. 备课资料管理
- 上传Maya相关资料（支持MA/MB、FBX、OBJ、PDF、PPT等格式）
- 资料分类与标签（按软件版本、功能模块等）
- 资料预览功能（支持图片、PDF等）
- 资料搜索与筛选

### 3. 理论教学支持
- 富文本编辑器，支持插入代码、公式
- 理论知识点整理与关联
- 教学案例库
- 常见问题解答管理

### 4. 多媒体支持
- 视频教程管理（支持本地视频和嵌入外部视频）
- 动画演示文件管理
- 截图和参考图片库

### 5. Maya专用功能
- Maya命令参考手册
- 快捷键指南
- 插件推荐与管理
- 渲染设置模板

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: Node.js + Express
- **数据存储**: 文件系统 (JSON格式)
- **部署**: 本地服务器

## 安装与启动

### 环境要求
- Node.js 14.0 或更高版本
- npm 6.0 或更高版本

### 安装步骤

1. **克隆或下载项目**
   ```bash
   # 克隆项目（如果使用Git）
   git clone <repository-url>
   
   # 或直接下载项目文件到本地
   ```

2. **安装依赖**
   ```bash
   cd maya-teaching
   npm install
   ```

3. **启动服务器**
   ```bash
   npm start
   ```

4. **访问系统**
   打开浏览器，访问 http://localhost:3001

## 使用指南

### 1. 系统登录
- 本系统为本地部署，无需登录即可使用
- 所有数据存储在本地文件系统中

### 2. 课程管理
- 点击导航栏的「课程管理」进入课程列表
- 点击「新建」按钮创建新课程
- 点击课程卡片查看课程详情
- 在课程详情页编辑课程信息和大纲

### 3. 资料管理
- 点击导航栏的「资料管理」进入资料列表
- 点击「新建」按钮上传新资料
- 支持拖拽上传文件
- 为资料添加分类和标签以便管理

### 4. 教学笔记
- 点击导航栏的「教学笔记」进入笔记列表
- 创建和编辑教学笔记
- 关联笔记与课程和资料

### 5. Maya工具
- 点击导航栏的「Maya工具」进入工具页面
- 浏览Maya命令参考手册
- 查看快捷键指南
- 了解推荐插件

### 6. 主题切换
- 点击导航栏右侧的月亮/太阳图标切换深色/浅色主题
- 主题设置会保存到本地

## API接口说明

### 课程相关
- `GET /api/courses` - 获取所有课程
- `GET /api/courses/:id` - 获取单个课程
- `POST /api/courses` - 创建课程
- `PUT /api/courses/:id` - 更新课程
- `DELETE /api/courses/:id` - 删除课程

### 资料相关
- `GET /api/materials` - 获取所有资料
- `GET /api/materials/:id` - 获取单个资料
- `POST /api/materials/upload` - 上传资料
- `PUT /api/materials/:id` - 更新资料
- `DELETE /api/materials/:id` - 删除资料
- `GET /api/materials/search` - 搜索资料

### 笔记相关
- `GET /api/notes` - 获取所有笔记
- `GET /api/notes/:id` - 获取单个笔记
- `POST /api/notes` - 创建笔记
- `PUT /api/notes/:id` - 更新笔记
- `DELETE /api/notes/:id` - 删除笔记
- `GET /api/notes/search` - 搜索笔记

### Maya工具相关
- `GET /api/maya/commands` - 获取Maya命令列表
- `GET /api/maya/hotkeys` - 获取快捷键列表
- `GET /api/maya/plugins` - 获取插件推荐
- `GET /api/maya/commands/search` - 搜索Maya命令

## 数据存储

- 课程数据: `data/courses/` 目录
- 资料数据: `data/materials/` 目录
- 笔记数据: `data/notes/` 目录
- 上传文件: `public/uploads/` 目录

## 常见问题

### 1. 服务器启动失败
- 检查Node.js版本是否符合要求
- 检查端口3001是否被占用
- 检查依赖是否安装成功

### 2. 资料上传失败
- 检查文件大小是否超过限制（默认100MB）
- 检查文件格式是否支持
- 检查存储空间是否充足

### 3. 页面加载缓慢
- 检查网络连接
- 检查服务器性能
- 减少同时上传的大文件数量

## 注意事项

1. **数据安全**
   - 本系统为本地部署，数据存储在本地文件系统中
   - 建议定期备份 `data/` 和 `public/uploads/` 目录

2. **文件大小**
   - 单个文件上传限制为100MB
   - 对于大型Maya场景文件，建议压缩后上传

3. **兼容性**
   - 支持主流浏览器（Chrome、Firefox、Safari、Edge）
   - 最佳体验分辨率为1920x1080

4. **性能优化**
   - 对于大量资料，建议合理分类和标签
   - 定期清理不需要的文件以保持系统性能
